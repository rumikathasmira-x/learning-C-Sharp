# C# Basics

## 📌 Introduction
C# is used to make interactive websites, mobile apps, video games, and desktop applications.

---

## 🖥️ Inputting and Displaying to the Console

| Method | Description |
|--------|-------------|
| `Console.WriteLine()` | Prints text to the console |
| `Console.ReadLine()` | Captures user input in the console |

---

## 💬 Comments
Comments are lines of code that are **ignored by your computer** — they're intended to be read by developers.

| Type | Syntax |
|------|--------|
| Single-line | `// your comment` |
| Multi-line | `/* your comment */` |
